# full-preload
A simple example of how to preload a HTML5 fully before playing it.

### Requirements
- Bash (Unix shell)
- [Node.js](http://nodejs.org/)

### Installation
1. Install dependencies.

```bash
npm install
```
    
2. Run.

```bash
npm start
```
